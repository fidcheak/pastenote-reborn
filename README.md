⚠️ **This is an educational project** created for learning and practicing modern frontend development concepts. It is not intended for production use.

## Features

- Clean, minimalist dashboard UI  
- Fast and responsive note management  
- Smooth UI animations and transitions  
- Focus on usability and visual clarity  
- Modern SaaS-inspired layout  

## Tech Stack

- **React** — frontend library  
- **Tailwind CSS** — utility-first styling  
- **Framer Motion** — animations and micro-interactions  
- **Create React App** — project bootstrapping  

## Getting Started

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn

### Installation (npm, on yarn also)

1. Clone the repository:
   ```bash
   git clone https://github.com/fidcheak/pastenote-reborn.git
2. Navigate to the project directory:
   ```bash
   cd pastenote-reborn
3. Install dependencies:
   ```bash
   npm install
4. Start the development server:
   ```bash
   npm start
The app will be available at http://localhost:3000
